## 初始化一个项目

- 使用npx：
    ```shell
      npx @omni-door/cli init -s myProject
    ```

- 若要创建多个项目，把脚手架安装到全局：
    ```shell
      npm i -g @omni-door/cli
    ```
  安装完成后执行初始化操作：
    ```shell
      omni init -s myProject
    ```
> note: 项目名称为 `myProject`

## 运行开发环境
  ```shell
    npm run dev
  ```

## 新建一个组件
  ```shell
    npm run new
  ```

## 构建项目
  ```shell
    npm run build
  ```
