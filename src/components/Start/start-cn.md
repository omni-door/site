## 初始化一个项目

1. 可以使用 `npx` 来初始化项目：
    ```shell
    npx @omni-door/cli init -s myProject
    ```

2. 若要创建多个项目，把脚手架安装到全局：
  ```shell
  npm i -g @omni-door/cli
  ```

  安装完成后执行初始化操作：
  ```shell
  omni init -s myProject
  ```

> 注意: 该项目的名称为 `myProject`

## 运行开发环境
  ```shell
  npm run dev
  ```

## 新建一个组件
  ```shell
  npm run new
  ```

## 构建项目
  ```shell
  npm run build
  ```

## 发布项目
  ```shell
  npm run release
  ```
